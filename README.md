## Requirements

1. Python 2.7.10
2. virtualenv

## Setup

Clone the repository:

    $ git clone git@github.com:brunitto/pork.git

Create the Python environment:

    $ cd /path/to/pork
    $ virtualenv ENV

Install requirements:

    $ pip install -r requirements.txt

Start the web application:

    $ bash start.sh
